PHP frameworky

Co je to framework?

Framework představuje jakousi základní kostru, na které může programátor stavět svůj projektu. Poskytuje pomocné utility, API, základní architekturu a řeší za programátora obecné problémy, se kterými se programátoři často setkávají.

Framework často pracuje na základě abstrakce, kterou programátor implemetuje a obohacuje ji o svojí byznys logiku, kterou jeho aplikace vyžaduje. Místo toho, aby se pak programátor zaobíral řešením obecných problémů, jako je třeba autorizace/autentizace, logování, přístup k databázi atd., může se soustředit přímo na svou aplikaci.

Dostupné PHP frameworky

PHP frameworků je na trhu opravdu mnoho. Podle průzkumu z roku 2019 těmi nejoblíbenějšími jsou Laravel, Code Igniter, Symfony a Zend.

Ostatní frameworky jsou například Yii, CakePHP, Fuel PHP nebo třeba Slim.
